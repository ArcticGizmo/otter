<h1 align="center">Otter</h1>
<p align="center">
 <img src="./src/icon.png" width="150" />
</p>

Automatically manages your Slack status based on what you're doing. Right now it sets your status when you're in a Microsoft Teams call — but that's just the first signal. The goal is hands-off Slack status management across all your tools.

```
dotnet run --project src\Otter.csproj
```
